# Secure Password Generator

## Live Demo
https://securepasswordgenerator1.netlify.app/

This is a secure password generator built with React. 
It allows users to generate random passwords with customizable length and character options such as uppercase letters, lowercase letters, numbers, and symbols.

## Features

- Generate random secure passwords
- Customize password length
- Include/exclude:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Copy to clipboard functionality

 ## Tech Stack

- React
- JavaScript
- CSS
- Netlify (Deployment)
